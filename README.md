# BongoBuy

**BongoBuy** is a Bangladeshi eCommerce platform built to make shopping smarter, simpler and user-friendly. It showcases products, features a live cart system and is fully responsive.

---

## Features

- User-friendly product browsing and filtering
- Cart system with live item count
- Responsive design for mobile and desktop
- Built with React, Tailwind CSS and React Router

---

## Tech Stack

- **Frontend:** React, Tailwind CSS, React Router
- **State Management:** Context API
- **Icons:** Lucide React
- **Bundler:** Vite
- **Styling:** Tailwind CSS
